## Rabbit App

系统介绍：模拟第三方平台，需要获取京东开放平台的授权，京东授权平台：https://gitee.com/CoderGeshu/jd_auth

**IDEA Tomcat 配置**

Open browser

- url: `http://localhost:8081/`

- VM options: `-Dfile.encoding=UTF-8`

Server Settings

- HTTP port: `8081`

- JMX port: `1099`
