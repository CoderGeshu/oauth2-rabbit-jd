## JD Auth

系统介绍：模拟京东开放授权平台，向 Rabbit 授权，Rabbit 项目地址：https://gitee.com/CoderGeshu/rabbit-app

**IDEA Tomcat 配置**

Open browser

- url: `http://localhost:8082/`

- VM options: `-Dfile.encoding=UTF-8`

Server Settings

- HTTP port: `8082`

- JMX port: `1010`
